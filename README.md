# 機電整合期末報告

<font>
本專題為大學課程「機電整合」之期末報告，題目為「模擬升降橫移式機械停車場」，模擬升降橫移式的機械式停車塔之操作。
藉由人機介面觸控板選擇停車位，再由PLC程式控制將指定車位移動到一樓後再開門，待操作結束後再關門並將車位復位。
